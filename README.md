## 🚀 Deployment (Render)

### Live Application

https://yazaki-avatar-sbgf.onrender.com

### Deployment Configuration

1. Push code to GitHub
2. Create a new Render Web Service
3. Connect repository:
   `m911development-hue/Yazaki-Avatar`
4. Runtime: Docker
5. Configure environment variables
6. Deploy

### Required Environment Variables

```env
OPENROUTER_API_KEY=your_key_here

APP_NAME=Yazaki AI

APP_URL=https://yazaki-avatar-sbgf.onrender.com

FRONTEND_URL=https://yazaki-avatar-sbgf.onrender.com

CHROMA_COLLECTION_NAME=yazaki_knowledge

CHROMA_PERSIST_DIR=/tmp/chroma
```
